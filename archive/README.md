# Archive

## `extension-old/`

Former location: `purefind-extension/`

This was the first cut of the PureFind Chrome extension. It was superseded by `/extension/` in early March 2026. Both dirs existed in parallel until the April 2026 audit, at which point this one was moved here to prevent drift.

**Do not load this in Chrome.** Use `/extension/` instead.

Kept only so the git history stays legible and we can diff if the new extension loses a feature.
